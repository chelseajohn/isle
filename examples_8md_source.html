3 Here are some example script showing how Isle can be used to perform HMC evolution and measurements.
4 
5 - \subpage fullHMCExample - HMC evolution with thermalization and production with hard coded parameters.
6 - \subpage thermalizationExample - Thermalize a configuration controlled through command line arguments.
7 - \subpage changeExample - Continue an existing HMC run but swap out the evolver.
8 - \subpage measureExample - Perform measurements on an ensemble of configurations.
9 
10 All scripts can be found in the directory `docs/examples`.
11 
12 They can all process basic command line arguments.
13 Use `py <scriptname> --help` to get an overview of the supported arguments.

42 \page changeExample Change Evolver
43 
44 This example demonstrates how to continue HMC production from an existing file
45 and change the evolver in the process.
46 It uses leapfrog integration together with shifts by 2*pi in the field variables to
47 jump over barriers in configuration space which would otherwise hamper ergodicity.
48 
49 This script requires the input file to contain checkpoints, not just saved configurations.
50 If this is not the case, more manual work is required to load / define the necessary objects
51 and set up the HMC driver.
