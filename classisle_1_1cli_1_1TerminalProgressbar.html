Detailed Description

A progress bar that is shown in the terminal via sys.stdio.

Needs to modify the current line of output in order to animate the progress bar. This is only possible when the output is indeed connected to a terminal. If sys.stderr is piped into a file, this class cannot operate properly.

Warning
Any and all terminal output must be made through TerminalProgressbar.write() while a progress bar is active! Otherwise it will interfere with the progress bar and might get erased when the bar is redrawn. When set up correctly (via setupLogging()), loggers handle output properly.
