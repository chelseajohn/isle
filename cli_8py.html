Detailed Description

Utilities for command line interfaces.

The default interface can be set up using isle.cli.init(). More control is available through the lower level functions.

The one function to control all the rest.

Initialize command line interface. This function must be called before any I/O as it sets up the logging framework.

Parameters
argParserCommand line argument parser. Can be
  • argparse.ArgumentParser: Use this parser as is.
  • str: Construct a parser based on this command name. See add*Args functions.
  • None: Don't parse any command line arguments. Log file and verbosity are set to the values provided in corresponding function parameters.
defaultLogLog file to use if argParser is None.
verbosityOutput verbisity level to use if argParser is None.
**kwargsPassed to isle.cli.makeDefaultParser() if argParser is a string.
Returns
Parsed arguments.

Parameters
targetTargeted number of iterations.
messageString to display with the progressbar.

def isle.cli.setupLogging (   logfile = None,
  verbosity = 0 
)

Set up Python's logging framework.

The root logger is set up to output to terminal and file. The former shows colored output if sys.stderr is connected to a terminal and is aware of Progressbar.

Parameters
logfileWrite log to this file. If None, no file output is performed.
verbositySet logging level. The minimum level for each handler is
verbosity terminal file
0 WARNING INFO
1 INFO INFO
2 DEBUG DEBUG
Exceptions
RuntimeErrorif this function is called more than once. It is safe to discard this exception, logging will still be set up properly.

Sets up and returns a new progress bar. The caller needs to advance that bar themselves.

messageMessage to display in front of the progress bar.
