Detailed Description

Abstract base class for progress bars.

Warning
Any and all terminal output must be made through TerminalProgressbar.write() while a progress bar is active! Otherwise it will interfere with the progress bar and might get erased when the bar is redrawn. When set up correctly (via setupLogging()), loggers handle output properly.
