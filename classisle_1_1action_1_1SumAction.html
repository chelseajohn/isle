Detailed Description

Sum of several actions.

Stores references to arbitrary instances of derived types of Action. Calling SumAction::eval() evaluates all actions and adds up the results; similarily for SumAction::force().

Attention
This is a view type. It stores references to the actions passed to it but does not own them. The user is responsible for freeing any resources appropriately.
