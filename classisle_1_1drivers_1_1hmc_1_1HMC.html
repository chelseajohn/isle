Detailed Description

Driver to control HMC evolution.

Stores metadata and current state of HMC evolution (apart from configuration and evolver). Manages generation of configurations and file output.

Note
Use isle.drivers.hmc.newRun() or isle.drivers.hmc.continueRun() factory functions to construct this driver and set up / load files.
