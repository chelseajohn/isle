A generator that behaves similarly to the builtin range with two differences: hingeRange

  • keeps going after reaching the end value and yields end forever and
  • accepts floats as parameters.
Parameters
startStart value (inclusive).
endEnd value (exculsive if called only int((end-start)/stepSize) times).
stepSizeSize of the steps while iterating from start to end. Must be negative if end < start.

Return a slice by replacing start or stop in the input if those are None.

Parameters
aSliceInput slice.
normStartStart value to use if inSlice.start is None.
normStopStop value to use if inSlice.stop is None. May be None, in which case size is used.
sizeNumber of elements described by the slice. If not None, it is used as normStop = normStart + size*inSlice.step.

Parameters normStop and size are mutually exclusive.

def isle.collection.subslice (   large,
  small 
)

Construct a slice for a sub collection.

Given an unknown collection of data full and a subset of it sub = full[large], construct and return a slice such that

sub[subslice(large, small)] == full[small]
Exceptions
ValueErrorif large and small do not allow for such a construction, e.g. small.start must be reachable from large.start in steps of size large.step.
