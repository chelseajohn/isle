◆ loadActionWeightsFor()

def isle.h5io.loadActionWeightsFor (   dset,
  base = "/" 
)

Load the weights from the imaginary part of the action for a measurement result.

The weights are loaded based on the 'configurations' attribute stored in the parent group of dset. This requires the attribute to be stored properly (no None) and the file to contain the values of the action for all the trajectories the measurement was taken on (can be as '/action/action' dataset or '/configuration' group).

The weights are computed as \(e^{-i S_{I}}\).

Parameters
dsetHDF5 dataset containing the measurement result.
basePath in HDF5 file under which the action is stored.
Returns
np.ndarray of the weights for dset.
