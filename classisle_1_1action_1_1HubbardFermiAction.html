Detailed Description

template<HFAHopping HOPPING, HFAVariant VARIANT, HFABasis BASIS>
class isle::action::HubbardFermiAction< HOPPING, VARIANT, BASIS >

Fermion action for Hubbard model.

The action is

\[ S_{\mathrm{HFA}} = - \log \det M(\phi, \tilde{\kappa}, \tilde{\mu}) M(-\phi, \sigma_{\tilde{\kappa}}\tilde{\kappa}, -\tilde{\mu}), \]

see HubbardFermiMatrixDia / HubbardFermiMatrixExp for the definition of M.

Both variants of the algorithm for are implemented and can be chosen in the constructor. The default is variant 1. See docs/algorithm/hubbardFermiAction.pdf for description and derivation of the algorithms.

This action can treat configurations as either in the spin or the particle/hole basis. This is controlled through the parameter alpha as

  • alpha == 0 - spin basis
  • alpha == 1 - particle/hole basis
Warning
Only supports nt > 2.

See docs/algorithm/hubbardFermiAction.pdf for description and derivation of the algorithms.
