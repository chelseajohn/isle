Detailed Description

Driver to perform measurements on configurations stored in a file.

Assumes that configurations are stored in the default format as written by e.g. the driver isle.drivers.hmc.HMC. Measurement results can be written to a new file or to the iput file. The driver checks for conflicts should the output file already exist and removes conflicting entries iff it is initialized with overwrite==True. This assumes that the measurements only write to the file under their savePath attribute and nowhere else.

Note
The driver should not be initialized directly, instead use isle.drivers.meas.init.

Apply measurements to all configurations in the input file and save results to the output file.

Makes sure that the measurements can be saved without conflicts. If entries with the same paths as measurement save paths exist in the file, they are deleted iff self.overwrite == True. Otherwise, a RuntimeError is raised.

Reads configurations from the input file in the order of the Markov chain. Compares the trajectory index extracted from file to the configSlice attribute of each measurement and if the index is in the slice, calls the measurement with that trajectory.

Warning
Calls Measure.mapOverConfigs with adjustSlices=True. See doc of that function.
Parameters
measurementsList of instances of isle.meas.measurement.Measurement to be called on each configuration in the input file.

Apply measurements to all configurations in the input file of this driver.

Reads configurations from the input file in the order of the Markov chain. Compares the trajectory index extracted from file to the configSlice attribute of each measurement and if the index is in the slice, calls the measurement with that trajectory.

Warning
If adjustSlices == True, the configSlice attribute of all measurements is modified to contain the actual trajectory indices read from the file instead of what the user specified (if those are incompatible, ValueError is raised). This is necessary in order to call Measure.save() later.
Parameters
measurementsList of instances of isle.meas.measurement.Measurement to be called on each configuration in the input file.
adjustSlicesModify the configSlice attribute of the measurements to reflect the configurations in the file.
